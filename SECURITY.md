# Security Policy

## Supported Versions
Security fixes are applied to the latest stable version on the default branch.

## Reporting a Vulnerability
Please report vulnerabilities privately (GitHub Security Advisory preferred).
Include affected version, steps to reproduce, and impact.

## Responsible Use
This project is intended for legitimate and lawful use.
Do not use it for unauthorized access, abuse, disruption, fraud, or illegal activity.
You are solely responsible for your use and legal compliance.
